# Agentic Design Patterns Index

The following patterns represent the fundamental building blocks for constructing sophisticated agents.

1.  **[Prompt Chaining](/design_patterns/prompt_chaining/)**: Breaking complex tasks into linear sequences of discrete, manageable steps.
2.  **[Routing](/design_patterns/routing/)**: Using conditional logic to dynamically select the most appropriate path or tool based on input context.
3.  **[Parallelization](#parallelization)**: Executing independent sub-tasks concurrently to optimize system efficiency and speed.
4.  **[Reflection](#reflection)**: Implementing feedback loops where agents evaluate and refine their own work to improve quality.
5.  **[Tool Use (Function Calling)](#tool-use-function-calling)**: Enabling agents to interact with external APIs, databases, and services to perform real-world actions.
6.  **[Planning](#planning)**: Decomposing high-level objectives into actionable multi-step strategies.
7.  **[Multi-Agent Collaboration](#multi-agent-collaboration)**: Orchestrating specialized agents to work together in hierarchical or networked structures.
8.  **[Memory Management](#memory-management)**: Maintaining conversational context (short-term) and persistent knowledge retention (long-term).
9.  **[Learning and Adaptation](#learning-and-adaptation)**: Allowing agents to evolve their behavior based on new data and experiential feedback.
10. **[Model Context Protocol (MCP)](#model-context-protocol)**: Standardizing how agents discover and communicate with external data sources and tools.
11. **[Goal Setting and Monitoring](#goal-setting-and-monitoring)**: Explicitly defining measurable objectives and tracking progress against them.
12. **[Exception Handling and Recovery](#exception-handling-and-recovery)**: Building resilient systems that can manage unforeseen errors and malfunctions.
13. **[Human-in-the-Loop](#human-in-the-loop)**: Integrating human judgment for safety, ethics, and nuanced decision-making.
14. **[Knowledge Retrieval (RAG)](#knowledge-retrieval-rag)**: Grounding agent responses in verifiable, up-to-date external information.
15. **[Inter-Agent Communication (A2A)](#inter-agent-communication-a2a)**: Facilitating standardized task delegation and collaboration between diverse agent frameworks.
16. **[Resource-Aware Optimization](#resource-aware-optimization)**: Dynamically managing computational and financial costs during agent operations.
17. **[Reasoning Techniques](#reasoning-techniques)**: Utilizing advanced methods like Chain-of-Thought and Tree-of-Thought to solve logical problems.
18. **[Guardrails/Safety Patterns](#guardrails-safety-patterns)**: Enforcing behavioral constraints and input/output filtering to ensure safe AI operation.
19. **[Evaluation and Monitoring](#evaluation-and-monitoring)**: Systematically measuring agent performance, efficiency, and compliance in live environments.
20. **[Prioritization](#prioritization)**: Ranking tasks and goals based on urgency, importance, and resource availability.
21. **[Exploration and Discovery](#exploration-and-discovery)**: Proactively seeking novel information and identifying "unknown unknowns" in complex domains.
